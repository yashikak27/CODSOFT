# Task 3 - ATM Interface

## Description

This project is a simple ATM Interface developed in Java. It allows users to check balance, deposit money, and withdraw money through a menu-driven console application.

## Features

* Check Account Balance
* Deposit Money
* Withdraw Money
* Insufficient Balance Validation
* User-Friendly Menu

## Technologies Used

* Java
* Scanner Class

## How to Run


```bash
javac ATMInterface.java
```


## Sample Operations

1. Check Balance
2. Deposit Money
3. Withdraw Money
4. Exit

## Learning Outcomes

* Menu-Driven Programming
* Conditional Statements
* Loops
* Basic Banking Logic
* User Input Handling
